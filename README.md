# telegram-FinanceBot
A simple Telegram Bot written in python to keep track of expenses in a group chat.

To use:
[x] Install requirements with `pip3 install -r requirements.txt`
[x] Fill out the `.ini.sample` file and rename to `FinanceBot.ini`
[x] Run with `./FinanceBot.py` or `python3 FinanceBot.py`

Run /help for help.
